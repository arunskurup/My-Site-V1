Installation
Clone the Repo:
> git clone https://github.com/arunskurup/My-Site-V1.git
> cd My-Site-V1
> composer install
> cp .env.example .env
> Set up .env file
> php artisan key:generate
> php artisan storage:link
> php artisan serve
http://127.0.0.1:8000/
